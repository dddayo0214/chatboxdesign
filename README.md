# 💬 ChatBox Design

一個線上的 **ChatBox / 對話框設計工具**，可以即時調整樣式，並自動產生 HTML + CSS。

## ✨ Features

- 🎨 即時預覽對話框
- 🌈 自訂背景顏色
- 🖤 自訂邊框顏色
- 📐 多種邊框樣式
- 🔵 圓角大小調整
- 🔠 字體大小與顏色調整
- 📏 自訂寬度、高度
- 🌫️ Box Shadow 即時調整
- 📋 一鍵匯出 HTML/CSS
- ⚡ 即時更新，不需要重新整理

## 🚀 Installation

Clone 專案

```bash
git clone https://github.com/dddayo0214/chatboxdesign.git
```

進入資料夾

```bash
cd chatboxdesign
```

安裝套件

```bash
npm install
```

啟動

```bash
npm run dev
```

Build

```bash
npm run build
```

## 💡 Why I made this?

每次設計遊戲或網頁的對話框時，都需要一直修改 CSS、重新整理畫面。

因此做了一個可以：

- 即時調整樣式
- 即時看到效果
- 一鍵複製 HTML/CSS

的小工具，希望能加快 UI 設計流程。
